  • <head> Container for meta data.</head>
  • <title> Contains title of document.</title>
  • <body> Contains the body of the page.</body>
  • <p> Contains a paragraph of text.</p>
  • <header> Container for links to other pages/top of page images </header>
  • <b> Bolds text</b>
  • <a href="link"> Links user to another page.</a>
  • <script> Can contain JavaScript </script>
  • <div> A section of the page. </div>
  • <span> Groups in-line elements. </span>
  • <strong> Bolds text. </strong>
  • <em> Italicizes text. </em>
  • <blockquote> Long quote. </blockquote>
