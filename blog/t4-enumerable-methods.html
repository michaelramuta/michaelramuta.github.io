Enumerables: Mapping

May 1, 2015

Enumerable is a module that allows for some great methods such as sorting, group_by, and cycle, but today we'll be talking about the map method.

Map lets us non-destructively modify a given array, and transform it in multiple ways. It essentially works like this:

array = [1,2,3]
array.map { |i| i }
=> [1,2,3]

In simple English, this code means that it is taking an array and applying a map to it, where for for each object in the array (i), it will equal i. This simply results in the original array. Here is another example for some other things we can do.

array = [1,2,3]
array.map { |i| -i }
=> [-1,-2,-3]

As you can see, for a given object i, we map it to negative i. There are quite a few ways to transform arrays with the map enumerable method. Here are a few more.

array = [1,2,3]
array.map { |i| i*i }
=> [1,4,9]

array = [1,2,3]
array.map { |i| "string" }
=> ["string","string","string"]

As you can see, changing up arrays with the map method can be pretty simple! Give it a try!
